== Description ==

Mail Lister provide an easy solution to marketing system,
you can easily save and collect customers or visitors emails 
and inside Mail Lister dashboard,you can send customized html5 emails 
to you customers,our goal is to provide simple way to marketing emails,
all you need is to customize your newsletter with html5 and css you can
even add images or gifs, when you done you can edit your shorcode, copy it
and paste it on the page you wish to display MailLister form.

* Easy to install 
* Easy to edit/manage
* Genrate more Leads
